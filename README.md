# ResolveAI

ResolveAI is a production-grade AI-first full-stack SaaS platform for support automation, RAG-based knowledge search, ticket triage, and agentic incident resolution workflows.

## Tech Stack

### Frontend

- React
- Next.js
- TypeScript
- Tailwind CSS

### Backend

- Node.js
- Express.js
- PostgreSQL
- Prisma
- Redis

### AI Service

- Python
- FastAPI
- LangChain
- LangGraph
- LangSmith
- RAG pipelines

### DevOps

- Docker
- Docker Compose
- GitHub Actions
- AWS

## Local Development

### Start database services

```bash
docker compose up -d
```
